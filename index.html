RESOURCES


Great websites to help you learn and sharpen your computer science skills. Click on each one to learn more about it and access its link (click on the .

Code



Code is a non-profit that is created to expand the learning of computer science by offering hour of code lessons, code studio, and local classes to anyone.

Coursera



Coursera allows anyone to take free college courses online in any topics, such as art, science, economics, mathematics, and of course, computer science. Online courses include videos presented by college professors. Some courses might include homework and test.

Khan Academy



Khan Academy provides interactive and online video for many subjects such as math, science, test prep, and computing. Must get an account to access their content.

Scratch



Scratch created by MIT Media Lab allows many to program interactive games and animations with simple comuting tools. It allows young people to think logically using only small tools to create something big.

TIPS ON APPLYING


-Be honest and be yourself.

-Do not procrastinate a couple of days before the deadline.

-Have someone proofread your essays and forms.

-Rejection can be a pain in the butt. Don't give up; there are many more out there.

-Do not apply to one just for the college application. Apply only for your learning and yourself.

-And most of all, GOOD LUCK!!! Love, Jozlin, Michelle, and Christy
